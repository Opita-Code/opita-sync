# Clasificación y control de salida

## Regla base
Todo artefacto nace restringido por defecto hasta que permisos, policies, delegación y contexto autoricen su acceso, uso, derivación o visualización.

## A qué aplica
- documentos
- registros
- dashboards
- resultados
- tools
- memoria
- trazas
- adjuntos
- exports
- artefactos derivados

## Clasificación base
La clasificación base aplica a todo artefacto del sistema y también a las acciones.

## Regla de salida
Si una salida completa no está permitida, el sistema debe devolver una versión redactada/parcial cuando esa salida parcial sí esté permitida.

## Implicaciones
La clasificación debe afectar:
- permisos
- delegación
- tools
- memoria
- analítica
- aprobaciones
- filtrado de resultados
