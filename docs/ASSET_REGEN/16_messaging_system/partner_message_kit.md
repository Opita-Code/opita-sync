# Partner message kit

## One-line description

Opita Sync is an AI-first governed control plane that helps organizations operate changes and workflows with approvals, evidence and recovery.

## Safe partner framing

- built on a governed kernel
- tenant-based activation
- focused on controlled operational value
- strong auditability and approval paths

## Partner warnings

Do not describe Opita Sync as:

- a universal AI engine
- a generic enterprise AI platform
- a fully autonomous operations agent
