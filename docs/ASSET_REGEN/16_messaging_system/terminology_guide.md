# Terminology guide

## Preferred terms

- governed control plane
- AI-first experience
- OSF-first authority
- tenant activation
- tenant operable
- governed corridor
- approvals, evidence, inspection, recovery
- grants and delegation governance

## Avoid or restrict

- universal engine
- interoperability engine
- enterprise AI platform
- autonomous execution layer
- free agent

## Clarification terms

- **OSF** = framework/kernel
- **Opita Sync** = product/control plane
