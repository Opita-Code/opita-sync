# Copy blocks

## Short description

Opita Sync is an AI-first governed control plane built on top of OSF.

## Medium description

Opita Sync helps organizations operate business changes and workflows with AI-first guidance while preserving approvals, evidence, inspection and recovery.

## Domain 1 block

Tenant configuration governance helps teams control what exists and is enabled for a tenant: capabilities, connectors and governance baselines.

## Domain 2 block

Tenant access and delegation governance helps teams control who can use, approve or delegate sensitive capabilities inside the tenant.

## Tenant activation block

Opita Sync creates value first by activating a tenant until it becomes operable, and then by supporting usage through the governed corridor.

## Governance block

Approvals, evidence, inspection and recovery are not optional extras. They are part of the product value.
