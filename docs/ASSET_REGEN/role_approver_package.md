# Role asset package — approver

## Role

- `approver`

## What this package is for

Use this package when the story is about:

- explicit approval
- sensitive changes
- decision traceability
- release / reject / escalate

## Primary promise

Help approvers make sensitive decisions with enough context, reason codes and traceability.

## Visual focus

- approval request
- preview signal
- release / reject / escalate
- approval-sensitive grants and delegations

## Best asset types

- approval journey slides
- governance-specific landing sections
- approval-sensitive scenario walkthroughs
- auditability messaging blocks

## Avoid

- making approval look like a lightweight click with no consequence
- hiding why approval exists in the corridor
