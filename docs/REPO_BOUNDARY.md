# Repo boundary

Este repo contiene **Opita Sync** como producto/control plane.

Incluye:

- source of truth del producto
- roadmap
- specs de productization gap closure
- definición de tenant operable
- narrativa y dirección del producto

No contiene el framework reusable como fuente primaria.

El framework vive en el repo hermano:

- **Opita Sync Framework**

Regla dura:

- Opita Sync consume OSF.
- Opita Sync no redefine runtime, truth plane ni policy core.
